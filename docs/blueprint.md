# **App Name**: Webapp Rápida

## Core Features:

- Formulario de Captación de Leads: Formulario integrado para recopilar información de contacto de los visitantes (nombre, correo electrónico, teléfono) para solicitar cotización, agendar llamada o contacto por WhatsApp.
- Presentación de la Propuesta de Valor: Mostrar de forma concisa y atractiva la propuesta de valor de Synerg-IA, destacando la velocidad de entrega, diseño moderno, enfoque en resultados y mínima fricción.
- Testimonios de Clientes: Incluir testimonios de clientes satisfechos con los servicios de Synerg-IA para generar confianza y credibilidad.
- Muestra de Portafolio: Mostrar ejemplos de landing pages creadas por Synerg-IA para que los visitantes vean la calidad del diseño y la efectividad en la conversión.
- Información de Contacto Directa: Mostrar claramente los datos de contacto de Synerg-IA (número de teléfono, correo electrónico, dirección) y enlace directo a WhatsApp para facilitar la comunicación.
- Información de Hosting/Dominio: Describe opcional hosting y registro de dominio (incluyendo registro en NIC.cl).

## Style Guidelines:

- Primary color: Deep teal (#008080) for professionalism and reliability.
- Background color: Light teal (#E0F8F8), desaturated for a clean, unobtrusive backdrop.
- Accent color: Turquoise (#40E0D0) for highlights and call-to-action buttons, drawing the eye.
- Body and headline font: 'Inter', a versatile sans-serif, ensures legibility and a modern feel.
- Use clean, professional icons to represent key services and benefits.
- Modern, minimalist layout with clear sections and plenty of white space to guide the eye.
- Include a 3D sphere (using Three.js or similar) in the header for a modern, eye-catching element.
- Subtle animations on scroll to highlight key sections and create a dynamic user experience.
